MEMEmory Game - Ghibli × Pixel Hybrid (Forest Cottage Theme)
This is a packaged version created for your project. Structure:

- index.php : start screen
- game.php : main game board
- action.php : AJAX backend endpoint
- result.php : winner screen
- style.css : theme (Forest Cottage)
- script.js : client logic
- classes/ : PHP OOP classes (Card, Deck, Game, Player, Difficulties)
- memes/ : placeholder meme images (per category). Replace with real memes.
- sounds/ : placeholder mp3 files. Replace with actual short sound effects.
- assets/ : small pixel wood texture used on card backs.

How to run:
1. Place the folder in a PHP-enabled server (XAMPP, MAMP, LAMP).
2. Ensure PHP sessions are enabled and the web server can read files.
3. Open index.php in the browser to start the game.
Note: Placeholder sound files are empty; replace them with small mp3 files.

Enjoy — Lloyd!
